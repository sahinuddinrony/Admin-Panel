
## Run project
Clone repository:
   https://github.com/sahinuddinrony/Full-Admin-Panel-Laravel.git
Go to folder:
    cd laravel-solid-example
Install dependencies:
    composer install
Copy .env file:
    cp .env.example .env
Generate app key:
    php artisan key:generate
Run migrations:
    php artisan migrate
